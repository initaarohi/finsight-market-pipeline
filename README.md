# FinSight Market Pipeline

A data pipeline project for financial market data ingestion, processing, and analysis.

## Project Structure

- **ingestion/** - Data ingestion scripts and configurations
- **glue-jobs/** - AWS Glue jobs for ETL processing
- **lambda/** - AWS Lambda functions for serverless operations
- **athena-queries/** - SQL queries for data analysis using Amazon Athena
- **ml/** - Machine learning models and related code
- **docs/** - Project documentation
